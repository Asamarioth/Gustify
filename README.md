# Gustify
Praca inżynierska.
Przedstawia dane wyciągne ze Spotify, takie jak rozkład gatunków muzyki w playlistach użytkownika oraz jego top 10 najczęściej słuchanych utworów.
Backend - API w C#
Frontend - React
